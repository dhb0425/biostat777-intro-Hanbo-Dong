This website was created using:
- **Quarto** - A scientific and technical publishing system
- **GitHub Pages** - Deployed from the `docs/` folder on the `main` branch
- **Theme** - Cosmo with custom CSS styling

To build the site locally, run `quarto render` in the project root directory.
